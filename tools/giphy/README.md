This tool allows the LLM to search for gifs on Giphy and display them in the chat. You can get started with a free Giphy developer account.

## How to use
1. Create a Giphy developer account here: https://developers.giphy.com/

2. Create an API key here: https://developers.giphy.com/dashboard/?create=true

3. Enter your API key in the tool's GIPHY_API_KEY valve
